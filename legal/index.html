Privacy Policy

This Privacy Policy explains how koopie ("the Bot") collects, uses, and discloses information when you use our services on Discord.

1. Data Collection

The Bot collects the following types of information:

  • User IDs: We collect User IDs of users who interact with the Bot. This is necessary for the Bot's functionality, such as identifying users for commands, permissions, or storing user-specific settings.
  • Guild (Server) IDs: We collect Guild IDs of the Discord servers the Bot is in. This is necessary to operate the Bot within those servers and to store server-specific configurations (e.g., custom prefixes, welcome messages, logging channel settings).
  • Channel IDs: We may collect Channel IDs for specific features, such as setting up logging channels, designated command channels, or announcement channels.
  • Message Content (Limited):
    • Direct Interactions: The Bot processes your message content when you directly interact with it (e.g., using slash commands, mentioning the bot, or sending messages in designated channels). This content is processed solely to fulfill your command or request and is generally not stored persistently unless explicitly required for a specific feature (e.g., if you use a "note-taking" command, the note content might be stored).
    • Logging/Moderation (Opt-in/Configurable): If a server administrator enables specific logging or moderation features, the Bot might process and potentially temporarily store message content, user IDs, and channel IDs for the purpose of maintaining server logs or enforcing server rules. This functionality is opt-in and configurable by server administrators.
  • Voice Activity (If Applicable): If the Bot joins voice channels (e.g., for music playback or voice moderation features), it may temporarily process voice data. This data is processed in real-time and is never recorded or stored by the Bot.
  • User-Provided Data (Opt-in/Configurable): The Bot may store data that users explicitly provide for specific features (e.g., custom role assignments, game statistics, reminder content, user-specific settings). This data is stored only with your consent through your interaction with the respective command.

2. How We Use Information

The information collected by the Bot is used solely for the following purposes:

  • To provide and operate the Bot's intended features and services.
  • To respond to your commands and requests.
  • To maintain server-specific configurations and settings.
  • To improve the Bot's functionality and user experience (e.g., error reporting).
  • For moderation and logging features, as configured by server administrators.
  • To comply with Discord's Terms of Service and Developer Policy.

3. Data Storage and Retention

  • User IDs, Guild IDs, and Channel IDs are stored for as long as the Bot is in a server or as long as a feature requiring their storage is active.
  • Message content processed for command fulfillment is generally not stored persistently.
  • User-provided data is stored until the user or a server administrator explicitly requests its deletion or removes the corresponding feature.
  • Data collected for logging/moderation features is retained as configured by the server administrator or for a limited period necessary for the feature's operation.
  • Voice data (if applicable) is processed in real-time and not stored.

4. Data Sharing and Disclosure

The Bot does not share any collected data with third parties, except in the following limited circumstances:

  • With Discord: Information is shared with Discord's API as necessary to operate the Bot (e.g., sending messages, managing roles).
  • Legal Requirements: If required by law or valid legal process.
  • Service Providers (If Applicable): If the Bot uses third-party services (e.g., a database provider, an external API for specific features), data may be temporarily processed by these services to fulfill the Bot's functions. We strive to use providers with strong data privacy practices. (e.g., Cloudflare)

5. Your Rights and Choices

  • Access and Deletion: You can request access to or deletion of your personal data stored by the Bot. Please contact our support server or email us at rekushii@icloud.com. For server-specific data, server administrators may have tools to manage or delete it.
  • Opt-Out: You can remove the Bot from your server at any time, which will result in the deletion of most server-specific data within 1 day.
  • Server Administrator Control: Server administrators have control over most data collected within their servers through the Bot's configuration commands.

6. Data Security

We implement reasonable technical and organizational measures to protect the information collected by the Bot from unauthorized access, alteration, disclosure, or destruction. However, no method of transmission over the internet or method of electronic storage is 100% secure.

7. Age Limitations

The Bot is intended for users who are at least 13 years old, in accordance with Discord's Terms of Service. We do not knowingly collect personal information from children under 13. If we become aware that we have collected personal information from a child under 13, we will take steps to delete such information.
